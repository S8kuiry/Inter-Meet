# Intermeet 🚀
**The Ultimate Technical Interview & Collaborative Coding Platform**

---

### 🌐 Live Project
**Experience Intermeet live here:** [https://inter-meet-client.onrender.com](https://inter-meet-client.onrender.com)

---

Intermeet is a robust, real-time platform designed to streamline technical interviews and collaborative coding sessions. It combines high-quality video conferencing with a powerful internal code editor, allowing users to practice coding problems solo or solve them together in live meetings.



## ✨ Features

### 💻 Collaborative Coding
* **Integrated Code Editor:** A fully functional, syntax-highlighted editor built directly into the platform.
* **Solo Practice Mode:** A dedicated space for developers to sharpen their skills on various coding problems.
* **Meeting Integration:** Solve complex algorithms in real-time with peers or interviewers during live video calls.

### 🎥 Video Conferencing
* **High-Quality Meetings:** Seamless video and audio powered by Stream API.
* **Real-time Interaction:** Chat and collaborate without leaving the interview environment.

### 🔐 Secure Authentication
* **Clerk Integration:** Enterprise-grade authentication and user management.
* **Protected Routes:** Secure dashboard and meeting rooms for authenticated users only.

### 🎨 Modern UI/UX
* **Sleek Design:** A professional bluish-green/teal themed interface designed for focus.
* **Responsive Layout:** Fully optimized for desktop and mobile browsers.
